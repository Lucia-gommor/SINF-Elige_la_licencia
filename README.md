# Juego indie: Pixel Play

¿Por qué eliges esta licencia?
He elegido esta licencia porque al tratarse de un juego indie, esta licencia facilita crear una comunidad y que el juego obtenga un mayor alcance.

¿Qué derechos das a otros?
Los derechos que doy a otros son: la posibilidad de modificar el código, que puedan añadir parches o modificaciones, que se puedan compartir versiones propias o modificadas, o que puedan analizar el código del juego para aprender o usarlo con otra finalidad.

¿Qué limitaciones impones?
Las limitaciones que impongo son: el deber de mencionarme como autora en el código en las versiones derivadas, y el uso de una licencia GPL-3.0 en sus derivados. Además, podría añadir alguna restricción aparte para controlar de forma más detallada los sonidos o gráficos originales, porque no tienen que estar sometidos a esta licencia.

¿Te conviene más a nivel económico y ético?
Me conviene más a nivel ético, ya que se permite crear un juego más accesible y transparente, así como que la comunidad se sienta incluida. Respecto al nivel económico, no tiene por qué ser totalmente gratuito, y al tener el apoyo de una comunidad se podrían fomentar las donaciones u otro tipo de aportes.
